# Pagos online

Acepta pagos en tu web o aplicación de forma segura con Sipay.

## Métodos disponibles

- Tarjetas (Visa, Mastercard, etc.)
- Bizum
- PayPal
- Apple Pay / Google Pay
- Pagos por banco
